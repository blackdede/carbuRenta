# Installation

```sh
pip install -r requirements.txt

python get_datas.py
python main.py
```

# Informations

Data is retrieved and stored locally via the get_datas script. This may take a few minutes

To get the display afterwards, run main.py and go to ```http://127.0.0.1:8050/```
